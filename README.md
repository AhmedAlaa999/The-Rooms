#The Rooms
## About The Game

The Roooms is a 2D platformer parkour game made using Godot.  
The game focuses on movement, timing, and challenging jumps across different rooms and obstacles.

## Features

- 2D parkour gameplay
- Multiple rooms 
- Pixel-style graphics
- Challenging platforming levels

## Controls

| Key | Action |
|---|---|
| A / D | Move |
| Space | Jump |

## Built With

- Godot Engine
- GDScript
